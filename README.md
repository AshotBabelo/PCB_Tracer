PCB Tracer — Ray-based Routing Algorithm (SBR)
PCB Tracer — это приложение на Qt/C++, реализующее модифицированный алгоритм Shooting and Bouncing Rays (SBR) для трассировки соединений на печатных платах.
Программа визуализирует процесс распространения лучей, находит кратчайший путь между компонентами и выполняет оптимизацию найденного маршрута.
